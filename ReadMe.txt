	COMMANDS: 

		* For compilation ONLY, use : 
			$ make

		* For execution ONLY, use :
			$ ./bin/EXEC

		* For compilation AND execution, use :
			$ ./run.sh


	INFORMATIONS:

		* WARNING:
			Make sure to have a terminal that can show at least 51 ligns and 145 columns !
			You can change the font size of your terminal by following these steps :
				- Open a terminal
				- click on : Edition > Preferences > Custom Font
				- Reduce the number for a full display of the parking

		* PERSONNALIZE YOUR PARKING :
			You can change user settings (like the price of the parking or the park min/max duration per vehicule) by editing "src/config.h". Feel free to
			personnalize your parking !
			You can also activate/desactivate the music in this file.

		* CONTROLS:
			By default, in the game mode "Play!", you can move your vehicule by holding 'z','q','s' and 'd'. That will make the vehicule goes up, to the left, down and to the right.
			You can modify thoses controls in the "src/config.h" with the ASCII code of the letter that you want to use.
			
			Link to the ASCII table : http://www.asciitable.com/
